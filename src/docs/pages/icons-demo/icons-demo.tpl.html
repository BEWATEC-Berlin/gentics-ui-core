Icons

The UI Core uses the Material Icons set. To use them, add an i element with the class .material-icons and then name the desired icon inside the i tag: 

<i class="material-icons">add</i>

Often we need the icon to align to the left or the right of some text or other content. In this case the .left or .right class should be added to the i element.
