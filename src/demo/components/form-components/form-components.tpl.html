Form Components

InputField

InputFields can be of type text, number or password. If no type is specified, it will default to text.

An InputField may have a label and/or a placeholder. If there is a label and no placeholder, the label will shift on focus. If there is both a label and a placeholder, the label will always be above the input.

Textarea

Like InputFields, Textareas can have a label and/or a placeholder, which interact in the same was as with InputFields. Textareas will also automatically grow to accommodate their content.

Checkbox

A Checkbox may have a label that is rendered right of the check box.
It can be mapped to boolean values or the value "indeterminate" with ngModel / ngControl / ngFormControl.

Check States: {{checkStates | json}}

RadioButton

A RadioButton may have a label that is rendered right of the radio input.
It can be mapped to values of any type (string, Object, number) with ngModel.

You can group muliple RadioButtons together by wrapping them in a RadioGroup.
The group can be attached to a form via the ngControl directive.
